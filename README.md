## 3. 요구사항 명세와 기능 명세
```mermaid
sequenceDiagram
    actor A as 사용자
    participant B as 블로그
    participant C as 서버

    A->>B: 회원가입
    B->>C: 사용자 정보 저장 
    A->>B: 로그인 요청 
    A->>C: 사용자 아이디와 비밀번호 전달

   alt 로그인 성공
        C->>B: 사용자 프로필 적용
        B->>A: 로그인 성공 메시지 출력
        C->>B: 메인페이지로 이동

    else 로그인 실패
        C->>B: 로그인 실패
        B->>A: 로그인 실패 메시지 출력

    Note over A, C: 사용자가 로그인 하였을때
        C-->>B: 전체 게시글 목록
        A->>B: 게시글 생성, 수정, 삭제
        A->>B: 댓글 작성
        B->>C: 게시글 및 댓글 생성,수정,삭제 반영    
    end
```
